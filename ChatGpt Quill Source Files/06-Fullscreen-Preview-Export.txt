What this category covers
-------------------------
This category covers the compose preview home surface, fullscreen preview editor, canvas settings, text layout, font and letter tuning, preview documents, export/share, saved images, and custom vs preset font rendering.

TextPreview inputs from App
---------------------------
- font: active FontSet.
- fonts: all FontSets for text-layer font choices.
- previewText and onPreviewTextChange.
- headerPreviewText and onHeaderPreviewTextChange.
- visibleHomeSections.
- selectedGlyph and spacebarGlyph.
- onApplyFontSpacing.
- onOpenCharacterEditor.
- onSelectCharacter.
- onSaveImage.
- onRecordExport.

Home visibility
---------------
TextPreview uses activeFont.homeSettings.visibleSections:
- exportControls controls the export heading, preset buttons, and Share/Export PNG actions.
- previewText controls Header and Body text inputs plus share status.
- If both are false, the panel uses minimal-preview-panel styling.

Fullscreen categories
---------------------
The bottom category tabs are:

- Add.

The source-level settings panels are:
- image.
- font.
- letter.
- position.

Canvas category
---------------
Controls canvas format, size, width, height, padding, page style, background, transparency, and manuscript controls. Manuscript uses parchment texture and extra sliders for age, edges, fibers, ink soak, ruling, and stains.

Font category
-------------
Controls preview-only font settings:
- size.
- width.
- height.
- letter gap.
- row spacing.
- spacebar.
- text effects.
- font/source choices for added text layers.
- Apply to persist font tuning.

Text effects
------------
Preview text effects are:
- texture.
- shadow.
- outline.
- smooth.
- thicken.

They are preview-only. They are composited through an offscreen text layer and affect header, body, and added text layers.

Letter category
---------------
Controls the selected glyph's preview-only metrics:
- size.
- width.
- height.
- baselineOffset.
- leftBearing.
- rightBearing.
- xAdvance.

It can open the selected glyph in GlyphEditor and can apply metric changes back to the saved font.

Position category
-----------------
Controls alignment and fit behavior. TextPreview has placement data for selectable text targets, so header/body/text layers can move independently.

Preview font rendering
----------------------
Custom fonts:
- build lines with measureCharacter and measureTextRun.
- draw through drawTextToCanvas and drawGlyph.
- use selectPreviewGlyph for variants.

Preset fonts:
- build lines with measurePresetTextRun and preset wrap helpers.
- draw with drawPresetTextToCanvas.
- apply backing FontSet metrics through getPresetCharacterMetrics.

Important current behavior: font and letter tuning works with preset fonts too. Do not patch only the custom glyph branch when fixing spacing, hit testing, or export layout.

Render flow
-----------
The main export path is:
1. getPhoneImageLayout computes fitted layout and text lines.
2. renderPhoneImageToCanvas sets canvas dimensions.
3. drawPreviewBackground draws the page.
4. drawPreviewContentToCanvas draws rules, text layers, and decor.
5. drawPreviewTextLayerWithEffects composites header/body/text effects.
6. drawPreviewDecorations adds doodles, stickers, and ornaments.
7. drawPreviewSelectionOutlines can draw editor selection targets in fullscreen.

Manuscript rendering
--------------------
For manuscript backgrounds, content is drawn to an offscreen canvas. drawManuscriptInkLayer multiplies, blurs, offsets, and interrupts ink according to manuscriptInkSoak so the text looks integrated into the page.

Preview documents
-----------------
Saved preview documents store:
- id.
- name.
- text.
- headerText.
- settings.
- textLayers.
- textTargetPlacements.
- updatedAt.

TextPreview stores up to 16 recent documents in localStorage under local-font-studio:preview-documents:v1.

Export and share
----------------
Browser:
- downloadPhoneImage converts the canvas to PNG Blob and clicks a link.
- save image can create a SavedImage data URL.

Native Capacitor:
- saveNativeFileToDocuments writes PNGs to Documents / Local Font Studio.
- shareNativeFile writes a cache file and opens Android sharing.

Saved images
------------
onSaveImage sends SavedImageDraft to App, which stores it through savedImageStorage.ts. SavedImagesPanel can later share, save, or delete the image.

Planning notes
--------------
- TextPreview is very large. Plan changes by category and render phase.
- Preview-only font/letter changes should not mutate FontSet until Apply is clicked.
- Preset and custom rendering must stay in parity for measurement, hit targets, export, and tuning.
- Fullscreen layout changes must be checked against the CSS bottom-panel sizing in styles.css.
