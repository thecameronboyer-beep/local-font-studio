What this category does
-----------------------
This category covers visual identity at two levels:
1. App themes, which change the Quill interface.
2. Font palettes, which change preview/export background, ink, accent, and texture for a font.

App themes
----------
App themes live in src\data\appThemes.ts.

Current AppThemeId values:
- classic.
- midnightCorrespondence.
- strawberryMarket.

Default app theme:
- classic.

App themes contain:
- id.
- label.
- description.
- paletteId.
- swatches.
- cssVariables.
- optional backgrounds.
- optional defaultBackgroundId.

Classic theme
-------------
Classic is the original dark studio look. It uses the original dark app background, gold primary accents, teal accent states, and warm light text.

Strawberry Market app theme
---------------------------
Strawberry Market uses:
- cream app background.
- berry primary buttons.
- cocoa ink.
- leaf green active states.
- seed gold decorative highlights.

Midnight Correspondence app theme
---------------------------------
Midnight Correspondence is based on:
- Parchment Ivory: #F5F1E8
- Moonlit Blue: #D7E3F4
- Dusty Slate: #A8B0BF
- Seal Garnet: #7A2E3A
- Antique Gold: #C9A45A
- Midnight Ink: #2B2F3A

It is meant to feel like private letters written at night, fountain pen and stationery, literary, thoughtful, calm, and confident.

Midnight app background variants:
- parchmentIvory.
- moonlitBlue.
- dustySlate.

App theme application
---------------------
App.tsx applies theme variables to document.documentElement:
- root.dataset.appTheme.
- root.dataset.appThemeBackground.
- root.style.setProperty for theme CSS variables.
- root.style.setProperty for selected background variant variables.

Theme choices persist in localStorage:
- local-font-studio:app-theme:v3
- local-font-studio:app-theme-background:v1

Planning notes
--------------
App themes and font palettes are separate on purpose:
- App theme changes the UI chrome.
- Font palette changes the output page and font identity.

When adding a new visual identity, usually create both:
1. an app theme in appThemes.ts.
2. a font palette in palettes.ts.

When adding a new preview background, update:
- BackgroundStyle type in fontTypes.ts.
- palette background definitions or TextPreview backgroundPresets.
- storage normalization in fontStorage.ts if needed.
- renderer/background CSS/canvas logic in TextPreview and GlyphEditor if it needs special drawing.
